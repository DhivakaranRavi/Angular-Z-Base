# Angular-Z-Base

This project is a base for PWA Angular from scratch with webpack configuration and fully customizable setup.

## Getting Started

```
git clone  https://github.com/DhivakaranRavi/Angular-Base.git
```

### Prerequisites

```
Npm -V^3.5.2
Node -V^8.10.0
```

### Installing

```
npm install
```

## Testing

```
npm test
npm run test:watch
npm run test:coverage
```

## Deployment

```
npm start
```

## Production

```
npm run build
```
